## Image Guidelines

1. **File formats:** Use JPG for photos, PNG for logos with transparency
2. **Optimization:** Compress images before uploading (use tools like TinyPNG)
3. **Naming:** Use lowercase, hyphens for spaces (e.g., `john-doe.jpg`)
4. **Aspect ratios:** Keep photos square (1:1) for consistency
5. **File size:** Keep individual images under 500KB for fast loading
